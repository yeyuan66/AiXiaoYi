# AI小益-收益中台运营助手 复刻全流程指南
## 一、前期环境与依赖准备（必须先完成）
### 1. 核心依赖安装（最新稳定版）
打开PyCharm终端，执行以下命令：
```bash
# 核心框架
pip install langchain==0.2.16 langchain-openai==0.1.23 langgraph==0.2.39

# 向量数据库ES客户端（匹配ES7.17.x）
pip install elasticsearch==7.17.14

# 意图识别
pip install fasttext==0.9.2

# SQL校验
pip install sqlglot==23.8.0  # 替代Java版JSqlParser，Python原生AST解析
pip install pyqlexpress==0.1.3  # QLExpress规则引擎Python绑定

# 工具与辅助
pip install python-dotenv==1.0.1 pydantic==2.9.2 redis==5.0.8
pip install numpy==1.26.4 pandas==2.2.3
```

### 2. 前置配置文件准备
在项目根目录创建`.env`文件，填入你的火山引擎GLM4.7配置：
```env
# 火山引擎GLM4.7配置
OPENAI_API_BASE=https://ark.cn-beijing.volces.com/api/v3
OPENAI_API_KEY=你的火山引擎API密钥
LLM_MODEL=glm-4-7b-32k  # 或glm-4-7b-128k，根据你购买的版本

# Elasticsearch配置（假设本地已部署ES7.17）
ES_HOST=http://localhost:9200
ES_USER=elastic
ES_PASSWORD=你的ES密码
ES_INDEX=finance_rag_knowledge_v1

# FastText意图模型路径
FASTTEXT_MODEL_PATH=./models/intent_model.bin

# 全局配置
MAX_RETRY_COUNT=2
MAX_EXECUTE_STEPS=3
CONTEXT_WINDOW_LIMIT=116000  # 输入Token上限，预留12K给输出
```

提前创建好项目目录：
